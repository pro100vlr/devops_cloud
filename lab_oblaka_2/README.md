Описание сервисов Microsoft Azure и их использование(в рамках данной работы Service Sub Type рассматриваются как варианты 
использования соответствующего Service Type и раскрываются в его описании.):

**Azure SQL Database**

- Управляемая реляционная база данных в облаке Microsoft Azure, предоставляющая возможности SQL Server без необходимости администрирования серверной инфраструктуры.

- Используется для хранения и обработки структурированных данных, выполнения SQL-запросов и обеспечения высокой доступности и масштабируемости баз данных.

**Azure Synapse Analytics**

- Аналитическая платформа для работы с большими объёмами данных, объединяющая хранилище данных и средства аналитической обработки.

- Применяется для хранения аналитических данных, выполнения сложных запросов и анализа данных в корпоративных системах.

**Azure Backup**

- Сервис резервного копирования данных и ресурсов Azure.

- Используется для создания и хранения резервных копий виртуальных машин, баз данных и других данных с целью восстановления при сбоях.

**Azure Site Recovery**

- Сервис аварийного восстановления и репликации данных.

- Применяется для обеспечения отказоустойчивости и восстановления работы сервисов при сбоях или авариях.

**Azure Storage**

- Облачное хранилище данных для хранения структурированных и неструктурированных данных.

- Используется для хранения файлов, таблиц, резервных копий и данных приложений.

**Azure Virtual Machines**

- Сервис предоставления виртуальных серверов в облаке.

- Применяется для запуска серверных приложений, служб и программного обеспечения, требующего полного контроля над операционной системой.

**Azure Machine Learning**

- Платформа для разработки, обучения и развертывания моделей машинного обучения.

- Используется для работы с данными, обучения моделей и выполнения вычислений в задачах искусственного интеллекта.

**Azure Batch**

- Сервис пакетной обработки вычислительных задач.

- Применяется для выполнения большого количества параллельных вычислений без необходимости ручного управления вычислительными ресурсами.

**Enterprise Licenses**

- Лицензии на использование корпоративного программного обеспечения в облачной среде.

- Используются для легального применения коммерческих продуктов в виртуальных машинах Azure.

**Azure DevOps**

- Набор инструментов для управления процессами разработки программного обеспечения.

- Используется для совместной работы команд, сборки, тестирования и развертывания приложений.

**Azure App Service**

- Платформа для размещения веб-приложений и веб-сайтов.

- Применяется для хостинга приложений без необходимости управления серверной инфраструктурой.

**Azure Data Factory**

- Сервис интеграции и управления процессами обработки данных.

- Используется для перемещения, преобразования и автоматизации обработки данных между различными источниками.
